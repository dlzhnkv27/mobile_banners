# Правила проекта Ivideon Mobile Banners

## Контент баннеров и пересборка

**Источник контента:** `banners-content.md`

При пересборке страницы контент баннеров берётся из `banners-content.md` и вносится в `index.html`.

Для каждого баннера в `banners-content.md` заданы поля:
- **Изображение** — URL/путь картинки баннера
- **Превью-заголовок** — заголовок в блоке баннера (справа от изображения)
- **Превью-текст** — краткий текст в блоке баннера (справа от изображения)
- **Полный заголовок** — заголовок в Sheet Modal
- **Полный текст** — текст в Sheet Modal (markdown)
- **Название кнопки CTA** — подпись кнопки в Sheet Modal

### Логика использования контента

1. **На карточке баннера (главная страница):**
   - `image` используется в `.banner-img img`.
   - `previewTitle` и `previewText` используются в `.banner-preview`.

2. **Внутри Sheet Modal:**
   - `fullTitle` заполняет `#sheetTitle`.
   - `fullText` заполняет `#sheetText` и рендерится как markdown:
     - `**текст**` -> жирный,
     - строки `- ...` -> список буллетов.
   - `ctaLabel` заполняет текст кнопки `#sheetCta`.
   - `image` используется в `#sheetBannerImage`.

3. **Отображение баннера в Sheet Modal:**
   - Подложка баннера в шите должна сохранять пропорцию `16:6` (как на главной).
   - Картинка в шите должна быть `height: 200%`, центрирована по горизонтали, с `top: 0`.
   - Нижняя часть картинки должна выходить за пределы видимой области баннера (обрезка через `overflow: hidden` контейнера баннера в шите).

**Что обновлять в index.html при пересборке:**
1. Разметка каждого `.banner-slide`: внутри `.banner-inner` использовать `image`, `previewTitle`, `previewText` из `banners-content.md`.
2. Массив `bannerData` в скрипте: для каждого баннера объект с полями `image`, `fullTitle`, `fullText`, `ctaLabel` из `banners-content.md`.
3. Логику рендера markdown в `#sheetText` и заполнения `#sheetBannerImage` не удалять при пересборке.

Итог: при изменении `banners-content.md` нужно выполнить пересборку и внести актуальный контент в `index.html` (разметка баннеров и массив `bannerData`).
