# Контент баннеров

Для каждого баннера заданы поля: изображение, превью-заголовок, превью-текст, полный заголовок, полный текст (markdown), название кнопки CTA.

---

## Баннер 1

**Изображение:** URL или путь к картинке баннера (16:6)

**Превью-заголовок:** Облачное видеонаблюдение

**Превью-текст:** Подключите камеры к облаку и смотрите видео из любой точки мира.

**Полный заголовок:** Облачное видеонаблюдение для дома и офиса

**Полный текст (markdown):**
```markdown
Подключите камеры к облаку Ivideon и смотрите видео **в реальном времени** из любой точки мира.

- Бесплатный пробный период 14 дней
- Облачное хранилище видео без ограничений
- Умные уведомления о движении 24/7
```

**Название кнопки CTA:** Подключить камеры

## Структура данных (для интеграции)

При подключении к CMS или JSON можно использовать такую схему:

| Поле | Тип | Описание |
|------|-----|----------|
| `image` | string | URL изображения баннера (пропорция 16:6) |
| `previewTitle` | string | Заголовок на карточке в карусели |
| `previewText` | string | Краткий текст на карточке |
| `fullTitle` | string | Заголовок в Sheet Modal |
| `fullText` | string | Полный текст в Markdown для Sheet Modal |
| `ctaLabel` | string | Текст кнопки CTA |
